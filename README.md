SystemVolumeControlwin7
=======================

classes that can be used to control system master volume 



Introduction
=======================

> volumeCtrlwin7 volume;
> // set mute
> setMute(true);  // true or false
> // set volume
> setVolume(80);  // range: 0.0~100.0
> // operator ++  / --
> ++volume;        // the step is define by system
> --volume;
